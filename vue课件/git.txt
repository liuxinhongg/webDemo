git 提交代码
1、git init 初始化
2、git add .提交所有的代码
3、git status 检查一下提交的代码文件
4、git commit -m “说明”
5、git remote add origin https://github.com/liuxinhongg/as.git
6、git push -u origin master
以上是将代码推送到git上面去


下面是更改了代码后提交
1、git add .
2、git commit -m "备注"
3、git pull origin master 第一次推送代码需要写
4、git push -u origin master
